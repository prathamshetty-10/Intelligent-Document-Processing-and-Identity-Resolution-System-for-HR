# Intelligent-Document-Processing-and-Identity-Resolution-System-for-HR
A document processing system that automates employee onboarding by extracting structured information from multiple HR documents such as resumes, Aadhaar cards, PAN cards, offer letters, appointment letters, and bank/personal information forms. The system intelligently merges data belonging to the same candidate using multi-field identity resolution, creating a unified employee profile without relying on a traditional database.

Use Case: Designed for HR teams to eliminate manual data entry during onboarding, reduce errors, and consolidate information from multiple documents into a single, structured record for faster employee processing.
